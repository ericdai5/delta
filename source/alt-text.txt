Figure 8: BNF-style grammar specification for the Delta configuration language. The grammar defines production rules for Config (top-level object containing formulas, variables, semantics, optional graph2d and graph3d arrays, and an optional stepping boolean), Formulas (array of objects with id and latex fields), Vars (array of variable declarations with optional default, input mode, range, step, name, and precision fields), Semantics (a JavaScript function receiving vars, sample, step, and latex helpers), Graph2D and Graph3D (graph configuration objects specifying axes, ranges, lines, points, and vectors with optional interaction, step-linkage, and persistence settings), and Labels (key-value pairs mapping formula selectors to display strings or LaTeX expressions). Selectors are LaTeX tokens identifying variables. Styling properties and the React component library are omitted.

Figure 9: Side-by-side JavaScript code listings illustrating schema duplication. Figure 10: Side-by-side code listings illustrating DOM and LaTeX manipulation patterns. The left listing (OLS Regression) shows React code that measures rendered tspan bounding-box geometry with getBoundingClientRect and getClientRects after each render, then manually repositions SVG highlight overlays using D3 translate transforms. The right listing (Why Momentum Really Works) shows code that caches pre-rendered MathJax HTML fragments in a hidden DOM element, retrieves them via querySelector, and performs string replacement on internal span markup to swap in dynamic numeric values.


Figure 11: Side-by-side code listings illustrating custom state-management patterns. The left listing (Backprop Explainer) shows a deeply nested React state object mixing UI flags (playButton), data arrays (X, y), hyperparameters (learningRate, epochs, loss, speed), and tunable parameters (m, b), with an async onClick handler toggling state and calling a method. The right listing (Eigenvectors and Eigenvalues) shows an AngularJS scope object storing mutable position and basis vectors, a deep $watch triggering manual recomputation of derived quantities, and D3 drag-behavior wiring through $apply calls.
